4.1

In this section we will talk about forms and inputs. Forms are used when we want to take data from the user.

Text input is used for normal text like name or username.

Password input is used for passwords and it hides the letters while typing.

Email input is used when we want the user to enter an email.

Number input is used for numbers only.

Date input lets the user choose a date.

Radio input is used when the user must choose one option.

Beginner Advanced

Checkbox input is used when the user can choose more than one thing.

HTML CSS

File input is used to upload a file from the device.

4.2

Now let's explain some elements that we use inside forms.

Label gives a name for the input so the user can understand what to write.

Select is used to make a drop down list.

Textarea is used when the user needs to write a longer text.

Button is used to create a clickable button.

Fieldset is used to group related inputs together, and legend gives this group a title.

Login Info

Datalist gives suggestions while the user is typing.

Option is the choices inside select or datalist.

Optgroup is used to group options inside the select list.

4.3

Form attributes controls how the form works.

Action is the place where the form data will be sent.

Method is how the data will be sent, like get or post.

Target controls where the result will open after submit.

Autocomplete helps the browser remember information the user wrote before.

4.4

Now lets talk about some input attributes.

Value gives the input a default value.

Readonly means the user can read the value but can not change it.

Disabled means the input is not active and the user can not use it.

Size changes the width of the input.

Maxlength controls the maximum number of letters.

Min and max are used with numbers to put a limit.

Multiple allows the user to choose more than one file.

Placeholder shows a small hint inside the input.

Required means this input must be filled before submitting the form.
